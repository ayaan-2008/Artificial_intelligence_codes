# AI Assistant

An advanced AI assistant built using Python and NLP.

## Features

- Open desktop applications
- Open websites
- Google search integration
- Weather detection by city
- Wikipedia knowledge engine
- Greeting recognition
- Multi-command execution
- Natural language understanding

---

## Technologies Used

- Python
- spaCy
- requests
- wikipedia-api
- subprocess
- webbrowser

---

## Installation

Install required libraries:

```bash
pip install spacy requests wikipedia-api
```

Install spaCy model:

```bash
python -m spacy download en_core_web_sm
```

---

## Run The Assistant

```bash
python AdvChatbot.py
```
